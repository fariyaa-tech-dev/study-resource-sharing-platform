# Study Resource Sharing Platform

A simple and attractive web platform where students can upload, share, search, and download study resources easily.

## Features

- User Registration
- Login and logout
- Upload Study Materials
- Share PDF Files and Useful Links
- Search Resources by Subject or Keyword
- Responsive and Modern UI Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
